# NEWT MCP

<!-- TODO: ヒーロー画像/GIF をここに配置（デザイナー作成中） -->

[newt.net](https://newt.net) のツアー・ホテル・目的地・空港検索を、Claude などの AI アシスタントから直接呼び出せる MCP（Model Context Protocol）サーバーです。

> ホスト型のため、サーバーを自前で立てる必要はありません。プラグインを追加するだけで利用を開始できます。

---

## できること

- **ツアー** — 検索、詳細情報、出発日カレンダー、料金、在庫、ツアー URL、写真
- **ホテル** — 検索、詳細情報、地域検索、料金、部屋、在庫、ホテル URL、写真
- **目的地・空港** — 目的地検索、空港検索、発着地の指定

すべて [newt.net](https://newt.net) のリアルタイム在庫・料金を参照します。
ロケール対応: `ja`（既定）、`ko`、`en-US`、`zh-Hant-TW`、`zh-Hant-HK`、`zh-Hans-CN`

---

## 使い方の例

MCP は AI アシスタントに「ツール」を提供する仕組みです。既存のチャットやワークフローに NEWT MCP を組み込むことで、こんな使い方ができます。

### 1. 社内専用の出張アシスタントを Slack でつくる

Slack の bot として Claude + NEWT MCP を呼び出せば、社員が「来週水曜から金曜の福岡出張、駅近で 1.5 万円以下のホテルを 3 つ」と聞くだけで候補が並びます。Notion の出張規程ドキュメントを併読させれば、予算上限・推奨エリアなど自社規程に沿った提案まで自動化できます。

### 2. 個人の「旅のコンシェルジュ」として Claude Desktop で常用

Claude Desktop にカスタムコネクタとして登録するだけで、家族の予定（Google カレンダー）や予算をふまえた旅行計画を会話で完結。「次の連休 4 日間の海外旅行候補を 3 つ、12 万円以内で」のような曖昧な依頼から、ツアー比較・日程組み立て・しおり下書きまで一気通貫で進められます。

### 3. 多言語インバウンド向け旅行プランナー

`en-US` / `ko` / `zh-Hant-TW` / `zh-Hant-HK` / `zh-Hans-CN` のロケールに対応しているため、訪日外国人向けのチャット窓口・LP・観光協会のコンシェルジュ機能などに組み込めば、利用者の言語のまま国内ツアー・ホテルを案内できます。

### 4. 旅行メディアの執筆・リサーチアシスタント

トラベルライターやメディア編集者が、最新の料金・人気エリア・ホテル設備などを Claude に問い合わせ、記事ドラフトや SNS 投稿の下書きを生成。手作業のサイト巡回が不要になり、調査と執筆を AI 中心のワークフローに置き換えられます。

## インストール

### Claude Code

```bash
/plugin marketplace add reiwa-travel/newt-mcp-server
/plugin install newt-mcp@newt
```

### Claude Desktop

カスタムコネクタとして以下の URL を追加してください。

```
https://mcp.newt.net/mcp
```

または `claude_desktop_config.json` に直接記述します。

```json
{
  "mcpServers": {
    "newt": {
      "type": "http",
      "url": "https://mcp.newt.net/mcp"
    }
  }
}
```

### Cursor / その他の MCP 対応クライアント

サーバー URL（Streamable HTTP）を指定してください。

```
https://mcp.newt.net/mcp
```

## プロンプト例

- 「来月のバリ 4 日間のツアーを 15 万円以下で探して」
- 「8 月 1 日からソウル出張、2 泊 3 日、エコノミークラスで探して。予算は 10 万円で。」
- 「来週末の京都のホテルを 5 つ比較してください」
- 「来週の大阪出張、駅近で評価の高いホテルを教えて」
- 「12 月の沖縄旅行、家族 4 人のプランを組んで」

## 利用可能なツール

### ツアー

| ツール | 説明 |
|--------|------|
| `tour_search` | 目的地・空港・日程・価格・座席クラス・特徴でツアーパッケージを検索 |
| `tour_detail` | ツアー詳細（プラン・フライト・ホテル・食事・行程） |
| `tour_detail_calendar` | 出発日・帰着日カレンダー |
| `tour_quotation` | 指定日の正確な価格、または日付なしでの価格レンジ |

### ホテル

| ツール | 説明 |
|--------|------|
| `hotel_search` | 日程・部屋数・地域で空室付きホテル検索 |
| `hotel_detail` | ホテル詳細（説明・設備・レビュー・地図） |
| `hotel_region_search` | 地域検索（都市・エリア・名所） |
| `hotel_region_detail` | 地域詳細（親・子・兄弟地域を含む） |

### 目的地・空港

| ツール | 説明 |
|--------|------|
| `destination_search` | 目的地検索 |
| `destination_detail` | 目的地詳細 |
| `airport_search` | 空港検索 |
